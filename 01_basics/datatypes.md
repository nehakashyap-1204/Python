# Object Types / Data Types

- Number : 1234, 3.1415
- String : 'spam', "Bob's"
- List : [1, [2, "three"], 4.5], list(range(10))
_ Tuple : (1, 'spam', 4, 'U')
- Dictionary : {'food' : 'spam', 'taste' : 'yum'}
- Set : set('abc'), {'a', 'b', 'c'}
- File : open('eggs.txt'), open(r'C:\ham.bin', 'wb')
- Boolean : True, False
- None : None
